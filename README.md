# Nail Club Tbilisi — Premium Landing Page ✨

A production-ready, $10,000 agency-level web application designed for **Nail Club Tbilisi**. Offering luxury nail care, laser treatments, eyebrow styling, and professional makeup services in Tbilisi, Georgia.

## 💎 Features

- **Multi-language Support**: Instant switching between Georgian (KA - Default), English (EN), and Russian (RU).
- **Theme Switcher**: Smooth transitions between luxurious Dark Mode and pristine Light Mode.
- **WhatsApp Conversion Engine**: Interactive booking form that extracts client requirements and transforms them into pre-filled WhatsApp click-to-chat messages in the user's selected language.
- **Micro-Interactions & Parallax**: Powered by **Framer Motion** and **TailwindCSS** for velvety smooth scroll reveals, animated badges, glassmorphism headers, and floating action buttons.
- **SEO & Accessibility**: 100/100 Lighthouse targeted architecture with semantic HTML5, WCAG AA compliance, ARIA attributes, structured JSON-LD schema, and Open Graph meta tags.

## 🚀 Quick Start

Follow these instructions to run the application locally or deploy to production:

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Open your browser at `http://localhost:5173` to view the live application.

### 3. Build for Production
```bash
npm run build
```
This command compiles TypeScript and bundles optimized assets into the `dist/` directory.

### 4. Preview Production Build
```bash
npm run preview
```
Test the production build locally before deploying to your hosting provider (Vercel, Netlify, Cloudflare Pages, etc.).
